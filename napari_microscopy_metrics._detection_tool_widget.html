class napari_microscopy_metrics._detection_tool_widget.DetectionParametersWidget(viewer)[source]

Bases: QWidget

A napari widget form for PSF detection parameters. It contains a DetectionToolWidget for setting detection parameters, a ThresholdWidget for setting threshold parameters and a RoiWidget for setting region of interest parameters. It is used in DetectionToolTab widget to create a parameters window and send parameters to the detection tool when applying detection.
