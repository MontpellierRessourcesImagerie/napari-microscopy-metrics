How To

This plugin is used to run full analysis of the PSF. This analysis is composed with :

  • Acquisition phase : The user have to enter parameters of the acquisition, those will be used massively, so it is a really important step.

  • Detection phase : The program will detect beads in image and extract region of interest by filtering beads to close from each other.

  • Metrics phase : It uses extracted beads to compute a gaussian fit to estimation full-width half maximum. Then calculates every metrics for analysis.

  • Outputs : It then produces many medias about analysis result to help user make his conclusions.
