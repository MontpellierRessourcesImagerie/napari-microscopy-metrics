Napari microscopy metrics documentation

Napari microscopy metrics is a Python plugin for Napari that gives tools to do the metrology of microscopic images using Point spread function (PSF). This plugin enables you to characterize optical aberrations, evaluate system resolution, and generate detailed reports for quality control in microscopy.

Whether you’re a researcher, engineer, or developer, Napari Microscopy Metrics allows you to analyze, visualize, and optimize your microscopy data on Napari.
