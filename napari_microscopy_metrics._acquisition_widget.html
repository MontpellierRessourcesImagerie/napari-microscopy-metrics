Acquisition widget

class napari_microscopy_metrics._acquisition_widget.AcquisitionToolPage(viewer)[source]

Bases: QWidget

A napari widget form for microscope acquisition parameters. It contains an ImageSizeWidget for setting scale informations and a MicroscopeParametersWidget for setting microscope informations. It also update scale informations in detection widget and napari viewer when applying a new scale.

viewer

The environment where the widget will be displayed.

Type:

napari.viewer.Viewer

countWindows

A counter to keep track of the number of windows created.

Type:

int

labelShape

A label to display the shape of the currently selected image layer

Type:

QLabel

initUi()[source]

A method to initialize the widget interface.

onLayerChanged()[source]

A method called when changing active layer. It updates scale informations in detection widget if the new active layer is an image and updates label with the shape of the new active image.
